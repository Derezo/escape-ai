Installing the APK

  1. Tap Download APK above and let the file finish downloading.
  2. Open it. Android will ask to allow installs from this source (your browser) — this is normal for an app outside the Play Store. Toggle it on, then go back.
  3. Tap Install, then Open. You'll land on the title screen — pick a name and a species and you're in.

The app bundles the game and connects to the live server at escape.mittonvillage.com — so you'll need a network connection, and other players you see are real. Nothing is uploaded from your device beyond the game's normal multiplayer traffic.
